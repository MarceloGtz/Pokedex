## Features

- Filter by type
- Searchbar for pokémon or pokémon id
- API consume

## Run Locally

Clone the project

```bash
  https://github.com/MarceloGtz/Pokedex.git
```

Go to the project directory

```bash
  cd Pokedex
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
